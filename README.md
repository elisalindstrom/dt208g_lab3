# DT208G Laboration 3
Webbplats skapad i Angular för att konvertera värden mellan olika enheter.

## Länk
Webbplats: https://dt208g-lab3.onrender.com

## Installation
Efter klonat repository installera dependencies genom npm install. Utvecklingsserver startas genom kommando ng serve.

## Användning
Webbplatsen består av flera komponenter: Home (startsidan), Convert (sida för konvertering), About (beskriver arbetet med uppgiften), Nav (navigationsmeny). Navigeringen hanteras med Angulars router-modul.

Värden som går att konvertera:
- Meter till Fot
- Fot till Meter
- Celsius till Fahrenheit
- Fahrenheit till Celsius
